# 🧠 SmartSaver Hub

SmartSaver Hub is a **multi-feature web application** designed to help users manage their finances efficiently. It combines three powerful modules into one platform:

* 💸 Expense Splitter
* 📊 Budget Planner
* 🛍️ Best Deals Finder

This project is built using **HTML, CSS, and JavaScript**, making it lightweight, fast, and beginner-friendly.

---

## 🚀 Features

### 💸 Expense Splitter

* Add and manage group members
* Split expenses among multiple users
* Track who paid and who owes
* Auto-calculate balances
* Update payment status (paid/unpaid)

---

### 📊 Budget Planner

* Add essential monthly items
* Track previous spending
* Set total budget manually
* Get smart budget suggestions
* Visualize spending using charts
* Editable product list via settings

---

### 🛍️ Best Deals

* Compare product prices across platforms
* View best discounts available
* Track price trends
* Search for products
* Save previous orders and savings
* Smart product comparison system

---
